# リッチメニュー設定仕様書

## 1. 概要
トリマーAIボットのLINEリッチメニュー設定に関する詳細仕様書

## 2. デザイン仕様

### 2.1 レイアウト
- **テンプレート**: タイル（小）
- **分割数**: 2分割（横並び）
- **サイズ**: 2500×843px（LINE標準）

### 2.2 ボタン配置
```
┌─────────────────┬─────────────────┐
│                 │                 │
│   💳 契約管理   │  📧 お問い合わせ │
│                 │                 │
└─────────────────┴─────────────────┘
```

## 3. ボタン仕様

### 3.1 契約管理ボタン（左側）
| 項目 | 設定値 |
|------|--------|
| **ラベル** | 契約管理 |
| **アイコン** | 💳 |
| **アクションタイプ** | テキスト |
| **送信テキスト** | サブスクリプション |
| **背景色** | #06C755（LINE グリーン）推奨 |

#### 動作詳細
1. ユーザーがボタンをタップ
2. 「サブスクリプション」というテキストが自動送信
3. ボットが以下を判定して応答：
   - **無料ユーザー**: 月額/年額プラン選択画面（Flexメッセージ）
   - **有料ユーザー**: 現在のプラン表示とプラン変更オプション（Flexメッセージ）

### 3.2 お問い合わせボタン（右側）
| 項目 | 設定値 |
|------|--------|
| **ラベル** | お問い合わせ |
| **アイコン** | 📧 |
| **アクションタイプ** | リンク |
| **URL** | Googleフォームの公開URL |
| **背景色** | #4169E1（ブルー）推奨 |

#### 動作詳細
1. ユーザーがボタンをタップ
2. 外部ブラウザでGoogleフォームが開く
3. ユーザーが問い合わせ内容を入力・送信

## 4. LINE Official Account Managerでの設定手順

### 4.1 リッチメニュー作成
1. LINE Official Account Managerにログイン
2. 「ホーム」→「リッチメニュー」を選択
3. 「作成」ボタンをクリック

### 4.2 基本設定
| 設定項目 | 値 |
|---------|-----|
| **タイトル** | メインメニュー |
| **表示期間** | 期間を設定しない |
| **メニューバーのテキスト** | メニュー |
| **メニューのデフォルト表示** | 表示する |

### 4.3 コンテンツ設定
1. **テンプレートを選択**: 
   - 「テンプレートを選択」をクリック
   - 「タイル（小）」カテゴリから2分割レイアウトを選択

2. **画像をアップロード**:
   - デザインに応じた画像（2500×843px）をアップロード
   - または、LINE Official Account Manager内でデザイン作成

3. **アクション設定**:

   **領域A（左側）- 契約管理**:
   - タイプ: テキスト
   - テキスト: `サブスクリプション`
   
   **領域B（右側）- お問い合わせ**:
   - タイプ: リンク
   - URL: `https://forms.gle/[GoogleフォームID]`
   - アクションラベル: お問い合わせ

### 4.4 保存と公開
1. 「保存」ボタンをクリック
2. 作成したリッチメニューの「･･･」メニューから「デフォルトに設定」を選択

## 5. デザインガイドライン

### 5.1 推奨カラー
- **契約管理ボタン**: #06C755（LINEグリーン）または #00B900
- **お問い合わせボタン**: #4169E1（ブルー）または #0084FF
- **テキスト色**: #FFFFFF（白）
- **境界線**: #E0E0E0（薄いグレー）

### 5.2 フォント
- **サイズ**: 32px以上（視認性確保）
- **太さ**: Bold（太字）
- **フォント**: Noto Sans JP推奨

### 5.3 アイコン
- 絵文字またはFont Awesomeアイコン使用
- サイズ: 48px以上推奨

## 6. 注意事項

### 6.1 画像作成時の注意
- 解像度: 2500×843px（必須）
- ファイル形式: JPEGまたはPNG
- ファイルサイズ: 1MB以下
- タップ領域を明確に区切る

### 6.2 運用上の注意
- リッチメニューの変更は即座に全ユーザーに反映
- A/Bテストを行う場合は、セグメント配信機能を使用
- 定期的にタップ率を確認し、必要に応じて改善
